## About JobiFy

JobiFy is a web application that serves as a platform for enterprises to post job offerings and users to apply and find employment opportunities. It simplifies the job search and application process, connecting businesses with potential candidates for a seamless and effective recruitment experience. It has been developed using Laravel,ReactJS,GraphQL.
